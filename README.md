Project Overview:

This project is an interactive Power BI dashboard designed to analyze customer behavior, credit card usage, and revenue trends. The dashboard provides insights into transaction activity, revenue performance, customer demographics, and satisfaction metrics.

The report enables stakeholders to monitor key financial and customer metrics and identify patterns across different customer segments.

The repository contains a Power BI Template (.pbit) file that allows users to connect their own dataset and generate the same analytical dashboard.

Objectives:

Analyze credit card transaction revenue

Understand customer demographics and spending behavior

Track weekly and quarterly revenue trends

Evaluate customer satisfaction scores

Identify high-performing customer segments

Dashboard Insights:

The dashboard provides the following key insights:

1. Revenue Analysis;

Total credit card revenue

Weekly revenue comparison

Previous week vs current week revenue trends

Interest earned from transactions

2. Customer Segmentation;

Revenue distribution by

Gender

Education Level

Income Group

Card Category

3. Transaction Insights;

Total transaction amount

Transaction trends by time (week, month, quarter)

Spending categories (Expense Type)

4. Customer Performance Metrics;

Customer satisfaction score

Active customers count

Revenue contribution by customer segments
